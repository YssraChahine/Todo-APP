# ToDo App (React)

A modern and responsive ToDo application build with React.
This project demonstrate core frontend development skills including state management, component based architecture, and local storage handling.

# Live Demo

[View Live App] (https://todo-app-seven-gamma-14.vercel.app/) 

## Features

- Add and delete tasks
- Mark tasks as completed
- Filter tasks (All / Active / Completed)
- Persistent data using LocalStorage
- Clean and modern UI with Styled Components
- Responsive design

## Tech Stack

- React (Hooks: useState, useEffect)
- JavaScript
- Styled Components
- HTML5 & CSS3
- LocalStorage API

## Installation & Setup

Clone the repository:

```bash
git clone https://github.com/(rest von dem Link einfügen)

Navigate into the project:
cd todo-app

Install dependencies:

npm install

Start the development server:

npm start

```
